分享商品到豆瓣东西
====================

- [在 Chrome Web Store 安装][webstore]
- [手动安装][download]

手动安装方法
------------

1. 下载[commodityshare-chrome][download]
2. 打开chrome浏览器，工具->扩展程序。或者直接在浏览器输入: `chrome://extensions`
3. 拖拽[commodityshare-chrome][download]到浏览器
4. Enjoy it.
